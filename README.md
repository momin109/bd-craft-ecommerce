# BD Craft Ecommerce

A full-stack ecommerce project inspired by Bangladeshi craft and lifestyle ecommerce platforms.

## Tech Stack

- Frontend: Next.js
- Backend: Node.js
- Database: MongoDB / MySQL / PostgreSQL

## Project Structure

```bash
bd-craft-ecommerce/
├── frontend/
└── backend/
```

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```

## Features

- Ecommerce homepage
- Product listing
- Product details
- Cart system
- User authentication
- Admin / backend API
